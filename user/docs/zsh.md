# zsh

## Requirements 

This configuration needs:

- [antibody]()
- [oh-my-zsh]()
- [screenfetch]() (TODO: make it optional)

## Installation
```bash
git clone https://github.com/Dzordzu/dotfiles.git
mkdir -p ~/.config/zsh
cp dotfiles/zsh/* ~/.config/zsh
cp dotfiles/zsh/.zshrc ~/.zshrc
rm -rf dotfiles
```
